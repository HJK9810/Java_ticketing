# TicketBox : 티켓팅 프로그램  
> 1. 이용권 종류, 시간, 주민번호 앞자리, 티켓수, 우대사항등을 입력 => 가격 출력
> 2. 계속하지 않을경우, 입력한 티켓들 출력
> 3. 최종 종료 직전, 숫자 0을 입력할 경우 데이터 분석 결과로 이동, 분석내용 출력
