### LotteTicketBox
* Ticketing.java : main  
* InputData.java : 데이터 입력 - scanner 사용  
* PrintUI.java : console에 데이터 출력  
* CalTickets.java : 모든 계산부분  
* SaveVals.java : 데이터 저장 & 파일에 데이터 입력  
* StaticValue.java : 모든 상수 저장 클래스  
* OrderData.java : 각 데이터(객체화) 저장클래스
